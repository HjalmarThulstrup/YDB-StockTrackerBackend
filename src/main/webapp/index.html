Stocks endpoints

GETStock
/api/stocks/single/{symbol}Returns a single stock with all its information
/api/stocks/list/{type}Returns a list of stocks with todays biggest gainers or losers
/api/stocks/news/{symbol}Returns a list of news for single stock

User endpoints

GETUser list
Protected endpoints
/api/user/{username}/listReturns users list of stocks
/api/user/{username}/symListReturn users list of stocks as symbols
POSTUser
/api/loginReturns a json object with a username and a json web token if username and password is correct
Protected endpoints
/api/createUserAdds a user to the database
/api/user/{username}/add/{stock}Adds a stock to the users favorites in the database
DELETEStock from user list
/api/user/{username}/remove/{stock}Removes stocks from users favorites in the database
